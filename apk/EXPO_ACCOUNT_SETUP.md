# How to Create Expo Account

## Method 1: Create Account via Website (Easiest)

1. **Visit:** https://expo.dev/signup

2. **Choose a signup method:**
   - **Email**: Enter your email and create a password
   - **GitHub**: Sign up with your GitHub account
   - **Google**: Sign up with your Google account

3. **Complete the signup:**
   - Fill in your details
   - Verify your email if required
   - You're done!

## Method 2: Create Account via CLI

1. **Run this command:**
   ```bash
   eas register
   ```
   
2. **Follow the prompts:**
   - Enter your email
   - Enter a username
   - Enter a password
   - Confirm your password

## After Creating Account

1. **Login via CLI:**
   ```bash
   eas login
   ```
   
2. **Enter your credentials:**
   - Email/Username
   - Password

3. **Start building:**
   ```bash
   eas build --platform android --profile production
   ```

## Important Notes

- Expo accounts are **FREE** for basic usage
- You get free cloud builds (limited per month)
- No credit card required for basic features
- Perfect for building APKs
